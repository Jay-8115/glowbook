Create a production-ready multi-vendor Hair Salon Booking Startup App using Flutter for mobile frontend, Node.js + Express backend, PostgreSQL database, and Firebase Authentication. The app must work smoothly on Android, iOS, and Web using Flutter.

Project Name: “GlowBook” (Salon Booking Platform)

Main Goal:
Build a startup-level salon marketplace app where:

Salon owners can register and manage their salons/services
Users can discover nearby salons using live location
Users can book salon services
Admin controls the entire platform
App UI should be modern, simple, fast, responsive, and startup-ready
